## ChatGPT/Claudeで関数を作る方法

### プロンプトの基本形

```
以下の条件でExcel関数を作ってください。

条件：
- A列に日付、B列に売上金額が入っています
- C列に、B列が10万円以上なら「優良」、5万円以上なら「良好」、
  それ未満なら「要改善」と表示したいです
```

このように**日本語で条件を説明するだけ**で、使えるExcel関数を返してくれます。

### よく使えるプロンプト例

{{< box type="info" title="コピペで使えるプロンプト集" >}}
**VLOOKUP の作成：**
「A列の商品コードをもとに、別シート「商品マスタ」のB列から商品名を取得するVLOOKUP関数を作ってください」

**条件付き集計：**
「A列が東京のデータのみのB列の合計を出すSUMIF関数を作ってください」

**日付計算：**
「A列の入社日から今日の日付までの勤続年数を年と月で表示する数式を作ってください」

**重複チェック：**
「A列のデータに重複があるか確認して、重複している場合に「重複」と表示する数式を作ってください」
{{< /box >}}

## VBAマクロをAIに作ってもらう

プログラミング不要で、**日本語でVBAマクロを作れます**。

プロンプト例：
```
以下の処理を行うExcel VBAマクロを作ってください。

処理内容：
1. Sheet1のA列のデータを読み取る
2. B列が「完了」のものをSheet2にコピーする
3. 処理が終わったらメッセージボックスで「完了しました」と表示する
```

生成されたコードをVBEエディタ（Alt+F11）に貼り付けるだけで使えます。

## Googleスプレッドシートの Gemini AI 活用

{{< box type="info" title="Googleスプレッドシートの組み込みAI機能" >}}
Google WorkspaceにはGemini AIが組み込まれており、スプレッドシート上で直接AIを使えます：

1. スプレッドシートを開く
2. 「ツール」→「Gemini」
3. 「売上データを分析して傾向を教えて」と入力

データをコピーしてChatGPTに貼り付ける手間なく、スプレッドシート上でAI分析が完結します。
{{< /box >}}

## ChatGPT Advanced Data Analysis 活用法

ChatGPT Plusでは**ファイルをアップロードしてAIに分析させる**機能があります。

1. ChatGPTに「Advanced Data Analysis」（旧Code Interpreter）を選択
2. ExcelファイルやCSVをドラッグ&ドロップ
3. 「このデータの傾向を分析して」「月別売上グラフを作って」と指示

AIがPythonコードを自動生成・実行し、**グラフや分析レポートを自動作成**してくれます。
