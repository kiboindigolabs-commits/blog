「ChatGPTを使ってみたけど、なんか微妙な回答しか返ってこない…」という経験はありませんか？実はその問題、AIの性能ではなく**質問の仕方（プロンプト）**に原因があることがほとんどです。

この記事では、誰でも今日から使えるプロンプトの書き方を、具体的な比較例とともに紹介します。

## プロンプトとは？

プロンプトとは、AIへの「指示文」や「質問文」のことです。同じAIでも、プロンプトの質によって返ってくる回答のレベルはまったく変わります。

{{< box type="point" title="プロンプトの重要性" >}}
優秀なAIも、あいまいな指示には曖昧な回答しか返せません。AIは「察する」能力がありません。明確に伝えることが最重要です。
{{< /box >}}

## 基本テクニック5選

### テクニック①：役割を与える（ロールプレイ）

AIに「あなたは〇〇です」と役割を設定すると、その専門家として回答してくれます。

| NG | OK |
|----|-----|
| マーケティングについて教えて | あなたはベテランのマーケターです。BtoB SaaS企業向けのコンテンツマーケティング戦略を教えてください |

役割設定の例：
- 「あなたは10年以上のキャリアを持つ人事のプロです」
- 「あなたはわかりやすい説明が得意な小学校の先生です」
- 「あなたは厳しいコードレビューを行くシニアエンジニアです」

### テクニック②：出力形式を指定する

「箇条書きで」「表形式で」「3つに絞って」など、欲しいフォーマットを最初に伝えましょう。

**例：**
```
以下の条件で回答してください。
・形式：箇条書き（5項目以内）
・文体：です・ます調
・難易度：IT初心者向け

質問：クラウドとは何ですか？
```

### テクニック③：前提情報・コンテキストを伝える

AIはあなたのことを何も知りません。背景情報を伝えると格段に精度が上がります。

**NG：**
```
ブログ記事のタイトルを考えて
```

**OK：**
```
以下の条件でブログ記事のタイトルを10個考えてください。
・ブログのテーマ：AI初心者向け情報メディア
・ターゲット：30〜50代のビジネスパーソン
・記事内容：ChatGPTを仕事で使うメリット
・SEOを意識した検索されやすいタイトル
```

### テクニック④：具体的な例（サンプル）を見せる

「こういう感じで」と例を示すと、AIはそのスタイルに合わせた回答をしてくれます。

```
以下のサンプルと同じトーンで、Claudeについての紹介文を書いてください。

【サンプル】
ChatGPTは、OpenAIが開発した対話型AI。難しい質問も、まるで友人に話しかけるように気軽に聞けるのが魅力です。ビジネスから趣味まで、あらゆる場面で活躍します。
```

### テクニック⑤：段階的に深める

一度の質問で完璧な回答を求めるより、会話を重ねて深めていく方が高品質な結果が得られます。

```
【1回目】
AIエージェントとは何か、概要を教えてください。

【2回目】
（回答を受けて）
その中で「ツール使用」の部分をもっと詳しく説明してください。
具体的なビジネス活用例も3つ挙げてください。
```

## プロンプトの型（テンプレート）

すぐに使えるテンプレートを紹介します。

### 文章作成テンプレート
```
【役割】あなたは優秀なコピーライターです。
【目的】〇〇（商品/サービス/記事）の文章を作成する
【対象読者】〇〇（年齢・職業・悩み）
【文体】〇〇（です・ます調/だ・である調/カジュアル）
【文字数】〇〇字程度
【その他条件】〇〇
```

### 要約テンプレート
```
以下の文章を要約してください。
・文字数：200字以内
・形式：箇条書き3点
・重要なキーワードは残すこと

【文章】
（ここに文章を貼り付け）
```

### アイデア出しテンプレート
```
【テーマ】〇〇についてのアイデアを出してください
【条件】
- 数：10個
- 実現可能性：低〜中（奇抜なアイデアも可）
- 前提：〇〇（業界・制約・ターゲット）
【評価軸】独自性・実用性・インパクトの観点でも短評を添えてください
```

## よくある失敗パターン

{{< box type="bad" title="これはNG！よくある失敗" >}}
- **あいまいすぎる**：「いい感じに書いて」→ AIは「いい感じ」が何かわからない
- **情報が多すぎる**：1回のプロンプトに10個以上の要求を詰め込む
- **否定形だけ**：「〜しないで」だけだと、何をすべきかが伝わらない
- **一発で完璧を求める**：まずドラフトを出してもらい、修正を重ねる方が効率的
{{< /box >}}

## まとめ

プロンプトのコツは「具体的に、明確に、コンテキストを添えて」の3点に尽きます。最初から完璧なプロンプトを書ける人はいません。試行錯誤しながら、自分なりの型を見つけていきましょう。
