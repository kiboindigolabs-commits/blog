## Midjourneyとは？

Midjourney（ミッドジャーニー）は、テキストの説明文（プロンプト）を入力するだけで**高品質なAI画像を生成**できるサービスです。

- 開発元：Midjourney, Inc.（アメリカ）
- 特徴：**芸術的・商業的品質**が業界トップレベル
- 利用方法：Discordのチャンネルで使用

DALL-E（ChatGPT）やStable Diffusionと比べて、**人物・風景・イラスト**の生成品質が特に高く、クリエイターや広告業界でも広く使われています。

## はじめ方（3ステップ）

### ステップ1：Discordアカウントを作る

Midjourneyは**Discord**を通じて使います。まだアカウントがない場合はDiscord公式サイトで無料作成。

### ステップ2：Midjourneyに登録

1. [midjourney.com](https://www.midjourney.com/) にアクセス
2. 「Sign In」→「Join the Beta」
3. DiscordアカウントでOAuth認証
4. 料金プランを選んでカード登録

### ステップ3：画像を生成する

Discordの「#newbies」チャンネルで：
```
/imagine prompt: 富士山と桜の風景、水彩画風、朝日
```

と入力してEnterを押すだけで画像が生成されます。

## プロンプトの基本

{{< box type="info" title="良いプロンプトの書き方" >}}
**[被写体] + [スタイル] + [雰囲気] + [技術的な指定]** の順で書くとGoodです。

例：
- `猫がコーヒーを飲んでいる、水彩イラスト、柔らかい色合い`
- `東京の夜景、ネオン、雨、映画的な構図、8K`
- `笑顔の女性ビジネスパーソン、プロフィール写真、白背景`
{{< /box >}}

### 便利なオプション

| オプション | 効果 | 例 |
|-----------|------|-----|
| `--ar 16:9` | 横長（YouTube用） | `--ar 16:9` |
| `--ar 9:16` | 縦長（スマホ・TikTok） | `--ar 9:16` |
| `--style raw` | よりリアルな描写 | `--style raw` |
| `--no text` | 文字を除外 | `--no text` |
