## 失敗①：AIの回答をそのまま信じる（ハルシネーション問題）

### よくある失敗

「ChatGPTが言っていたから正しいはず」と、回答を事実確認せずに使ってしまう。

### なぜ起きるのか

AIは**ハルシネーション（幻覚）**と呼ばれる現象を起こすことがあります。もっともらしい嘘を自信満々に答えてしまうのです。特に：

- 統計・数字・パーセンテージ
- 人物のプロフィール・経歴
- 最新のニュース・価格情報
- 法律・医療・税務の専門知識

これらは要注意です。

### 正しい使い方

{{< box type="warn" title="AIの回答は必ず一次情報で確認する" >}}
- 統計データ → 元の調査機関のサイトで確認
- 法律・医療情報 → 専門家や公式情報を参照
- 最新情報 → Perplexity AIや検索エンジンでダブルチェック
{{< /box >}}

## 失敗②：漠然とした質問をする

### よくある失敗

「いい文章を書いて」「マーケティングについて教えて」のような漠然とした指示を出してしまう。

### 正しい使い方

**5W1H**で具体的に指示を出しましょう：

```
❌ NG：「営業メールを書いて」

✅ OK：「以下の条件で営業メールを書いてください。
・送り先：中小企業の経営者（40〜50代男性）
・商品：クラウド型の経費精算ソフト
・目的：無料体験版の申し込みを促す
・文章量：400文字以内
・トーン：丁寧だが押しつけがましくない」
```

具体的な条件を与えるほど、期待通りの回答が返ってきます。

## 失敗③：機密情報・個人情報をAIに入力する

### よくある失敗

社内の機密情報、顧客の個人情報、社員名、未公開の財務情報などをAIに入力してしまう。

### なぜ危険なのか

{{< box type="bad" title="入力した情報は学習に使われる可能性がある" >}}
ChatGPTなど一部のサービスでは、入力したデータがモデルの学習に使用される場合があります。機密情報を入力すると情報漏洩のリスクがあります。
{{< /box >}}

### 正しい使い方

- 個人名・会社名は**匿名化**してから入力
- 機密情報は入力しない（架空の数字で代替）
- 企業利用は**APIモード**や**法人契約**（データ学習OFFが可能）を使う

## 失敗④：一度でベストな答えを求める

### よくある失敗

一度質問して「思ったのと違う」と諦めてしまう。

### 正しい使い方

**対話を重ねて改善**しましょう。AIは会話の流れを覚えているので：

```
最初：「このメールを修正して」
↓
2回目：「もう少し丁寧な表現にして」
↓
3回目：「最後の段落をもっと具体的にして」
↓
4回目：「全体を100文字短くして」
```

こうした対話型のやり取りで、理想の文章に近づけられます。

## 失敗⑤：AIを万能だと思いすぎる

### よくある失敗

「AIに全部やってもらえばいい」と丸投げしてしまい、品質チェックを怠る。

### AIの苦手なこと

{{< box type="bad" title="AIが苦手なこと" >}}
- リアルタイム情報（学習データに依存）
- 複雑な計算・統計分析（専用ツールが必要）
- 感情・共感が必要なコミュニケーション
- 自社独自の情報・ノウハウの生成
- 完全なオリジナリティ（類似表現が出ることがある）
{{< /box >}}

AIはあくまで「優秀なアシスタント」。最終的な判断・責任は人間が持つことを忘れずに。

## まとめ

{{< box type="good" title="AI活用の5つのポイント" >}}
1. **ハルシネーションに注意**：重要な情報は必ず一次情報で確認
2. **具体的な指示を出す**：5W1H でコンテキストを与える
3. **機密情報は入力しない**：個人情報・社外秘は匿名化
4. **対話で改善する**：一度で完璧を求めない
5. **最終確認は人間が行う**：品質責任は自分が持つ
{{< /box >}}

この5つを意識するだけで、AIの活用効果が大幅に上がります。失敗を恐れず、まず使ってみることが上達への近道です。
