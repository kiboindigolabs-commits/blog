## 無料AIツール10選

### チャット・文章系AI

{{< card name="ChatGPT（OpenAI）" stars="5" price="無料〜" url="https://chatgpt.com/" >}}
世界最多ユーザーのAIチャット。無料版でもGPT-4oが使える。文章作成・質問回答・翻訳など万能。
{{< /card >}}

{{< card name="Claude（Anthropic）" stars="5" price="無料〜" url="https://claude.ai/" >}}
長文作成に強く、自然な日本語が特徴。ブログ・メール・報告書作成に最適。無料版でも高品質。
{{< /card >}}

{{< card name="Gemini（Google）" stars="5" price="無料〜" url="https://gemini.google.com/" >}}
GoogleアカウントがあればすぐOK。Google検索連携で最新情報を参照。Googleサービスとの連携が強み。
{{< /card >}}

{{< card name="Perplexity AI" stars="4" price="無料〜" url="https://www.perplexity.ai/" >}}
AI検索エンジン。ソース付きで最新情報を答えてくれる。リサーチ・情報収集に最強。
{{< /card >}}

### 画像生成系AI

{{< card name="Adobe Firefly" stars="4" price="無料（25クレジット/月）" url="https://firefly.adobe.com/" >}}
Adobeのサービスなので商用利用が安心。テキストから画像・テキストエフェクトを生成。著作権リスクが低い。
{{< /card >}}

{{< card name="Microsoft Designer（Copilot）" stars="4" price="無料" url="https://designer.microsoft.com/" >}}
Microsoftアカウントがあれば無料で使える画像生成AI。Bing Image Creatorと同じ技術でDALL-E 3を使用。
{{< /card >}}

{{< card name="Canva AI" stars="4" price="無料〜" url="https://www.canva.com/ja_jp/" >}}
デザインツールCanvaに組み込まれたAI。背景削除・画像生成・テキスト生成がデザイン作業中にできる。
{{< /card >}}

### 音声・動画系AI

{{< card name="CLOVA Note（LINE）" stars="4" price="無料" url="https://clovanote.line.me/" >}}
LINEの無料文字起こしサービス。録音をアップするだけで話者識別付き文字起こしが完成。会議・インタビューに。
{{< /card >}}

{{< card name="Google NotebookLM" stars="4" price="無料" url="https://notebooklm.google.com/" >}}
Googleのドキュメント分析AI。PDFや動画URLを入れると要約・Q&A・ポッドキャスト生成まで自動でできる。
{{< /card >}}

{{< card name="DeepL翻訳" stars="5" price="無料〜" url="https://www.deepl.com/ja/translator" >}}
業界最高水準の翻訳精度。英語・日本語・欧州語に対応。月50万文字まで無料。Google翻訳より自然な訳が特徴。
{{< /card >}}

## 無料で使う際の注意点

{{< box type="warn" title="無料プランの制限に注意" >}}
- 多くのツールには**1日・1月の使用回数制限**がある
- 制限に達したら別のAIに切り替えて使うのがコツ
- 例：ChatGPT制限 → Claude → Geminiで代替
{{< /box >}}
